# 专业基础及必修课程

!!! info 

    将能院六个专业所涉及的主要专业基础课与必修课按照学科模块整理，譬如`力学类`模块包含`工程力学`、`工程流体力学`等课程，共分为**机械类**、**力学类**、**电学及控制类**、**数理类**、**能源与热学类**五个模块。
