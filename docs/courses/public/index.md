# 专业基础及必修课程

将能院六个专业所涉及的主要专业基础课与必修课按照学科模块整理，譬如`力学类`模块包含`工程力学`、`工程流体力学`等课程；
此外还将一些内容重复的课程合并在一起，如`电学及控制类`模块中`电工电子学及实验`包括了过控所涉及的`电工电子学`与`电工电子学实验`
