本人上的盛正卯老师的数理方法课，I的成绩并不好，II的时候吸取经验获得了4.8，在此分享我的经验，希望能够帮助到大家

# 课前提醒：
这门课真的是一门需要**信念感**的课程，内容确实比较难，但是盛老师非常和蔼，也非常的有耐心，讲课也很有条理（虽然一开始可能会听不懂），最最重要的是老师非常捞，一定要相信老师、相信自己。~~只要在试卷上写了东西就可以得到分~~

# 关于上课：
上课的内容客观上是比较难的，盛老师上课主要把重点放在一些推导上，因为内容确实比较多讲得会比较快，建议以**听明白优先、笔记有时间就记**的原则，课上紧跟老师的思路，可以课后看智云补笔记。

盛老师上课并不完全按照书上的内容，建议是另外记笔记不要记在书上。

# 关于作业：
可能上课听懂了，看到作业还是会云里雾里甚至是不知所措，网上确实有答案哈，但是没必要为了应付作业抄抄，就算完全不会做**看了答案之后自己做**出来也是很有帮助的，考试中有的题目就是作业的类似题。另一方面，其实网上答案有些一看就是错的还是要有些辨别能力，如果实在困难也可以问老师和助教，他们都非常热心回答的。

# 关于考试：
数理方法I有期中考试，98上可以找到一些理念卷，内容大差不差，主要是数学（复变）过程，重点要关注一下迪利克雷函数，他还特别喜欢考薛定谔方程相关的。老师说期中考试考得好就把占比调的高一点，如果考得不好就当作大家没有考过hhh

数理方法I的期中考试是前半复变的部分＋后面一点物理方程相关的部分，除了期中范围，比较重要的是物理泛定方程的建立过程和球函数，泛定方程的建立盛老师花了很长时间讲解，考试要求也不仅仅是几个比较普通的泛定方程；球函数考察的是球函数的推导，甚至是把同样的方法嫁接到别的坐标系中，这部分课上详细介绍了并且还额外布置了相关作业，如果不懂可以问问助教老师。

数理方法II是只有期末考试的，最最最最最重要的就是第九章的最后面的一个各种方程的特征函数的表格（实际是后面几章的总结），把这个表格搞明白就胜利了一大步。

虽然考试范围不同，但是共性是老师很重视书上的例**题、推导、甚至还有些作业**，复习的时候也不用特别刷题，把书上内容全搞明白基本就可以得到比较好的成绩了。**实在实在**不行了背点例题考试的时候对应着默上去说不定还有不少分（不到万不得已还是不要干了）
