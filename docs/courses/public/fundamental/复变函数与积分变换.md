# 复变函数与积分变换

## 课程内容

### 分数构成

作业与平时表现（30%）+小测（赋分）（20%）+期末（50%）

小测位于第四/五周

### 课程大纲

复变函数是高等院校理工科学生必须具备的数学知识，它是微积分的重要后续课程。内容包括解析函数、复变函数的积分、柯西积分公式、罗朗级数、保角角映射以及积分变换等。
