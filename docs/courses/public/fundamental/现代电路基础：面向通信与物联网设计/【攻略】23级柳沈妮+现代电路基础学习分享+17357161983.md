# 关于上课
上课一开始难度给人特别高的感觉，尤其是对于没有选技术的朋友，会感觉自己似乎少学了一门课，云里雾里的。但是往后学会发现这门课似乎**更想让你了解而不是熟练掌握**，更想让你见识一下这些技术而不是让你自己熟练使用这些技术。

学的时候一定要谨记这门课是**开卷（但是只能带课本）**，可以把笔记或者你觉得有用的东西全部记录在你的书上。
