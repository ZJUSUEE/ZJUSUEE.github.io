# 控制论

## 课程内容

### 分数构成

慧能预置的专业基础课程，但是对慧能不能算通核。需要另外选修一门通核课。

讨论课平时分50+期末50。

讨论课分小组展示和个人展示，一学期下来每个人有两次展示，讨论课发言满分大概需要十次。
