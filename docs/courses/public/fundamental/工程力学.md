# 工程力学

## 课程内容

### 先修要求
- 高等数学

### 分数构成

- 线下考核70%：平时作业占10%，出勤和课堂表现占5%，学习体会小论文占5%，期末考试50%；

- 线上考核30%：相关补充材料和视频观看占5%，线上讨论占10%，结构设计并二级加载成功占15%。

### 课程大纲

- 理论力学部分
- 材料力学部分
