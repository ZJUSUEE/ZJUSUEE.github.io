# 写在前面
这门课大家可以当作娱乐来学习，非常有意思，原地进厂。

98和朵朵上有很多人说这门课4.2和3.9两点分布，但是大家还是有必要认真学习的，lz本人很喜欢这方面的内容，认真学也拿到了4.8，包括lz的朋友们也有很多拿到了4.5，还是可以脱离两点分布的。

开学的安全测试98上有答案，可以第一次上课前就去下载下来。

# 关于预习
**一定要每周看看预习要求！！！！**这非常重要，尤其是那个线切割老师，基本上给你的打分就是你的预习成果，你不预习他会给你低分，全班不预习他就要准备开喷了。

# 关于课后作业
**预习作业和复习作业每周也一定要记得做！！！！**我记得还有几次测试。这些题目大差不差98上基本上都有，善用搜索；同时据说“大学搜题酱”app上也有，lz没有试过，但如果实在找不到大家也可以这么干试试看。但是不排除每一年多一些题目。

# 关于上课
**一定要记得带衣服和帽子！！！！**

最重要的就是老师讲的时候不要分心！！！不要分心！！！只要短暂的记住老师讲的话一个小时就可以（就是支撑自己做完）。如果大家想卷分数的话，可以做的花里胡哨一点，但是前提是在老师要求范围内。比如，3d打印一个牌子的时候可以增加一点图形啊什么的。

# 关于考试
最后一节课会组织一次复习课，可以听一听。复习的时候主要针对这个ppt和书本内容，但是多选题也会有平时的内容，比如识别代码。考试占比不是特别高，且全是选择题，对欧皇友好。

我记得考前有一次上机考试，认真做就好，占比非常低，大家都是40-60分，没啥区分度。
