# 工程热力学

> 2026年2月更新

## 课程内容

### 分数构成

到课率和课堂讨论表现占 20%，课外阅读报告、平时作业、课程论文占20%，阶段性考试和期终考试占60%。

### 课程大纲

主要内容包括四大部分：

- **第一部分**为热力学基本定律，包括基本概念及定义、能量与热力学第一定律、熵与热力学第二定律等。
  
- **第二部分**为工质的热力性质，包括一般热力学关系和气体的热力性质。

- **第三部分**为热力过程及热力循环，包括气体的压缩、蒸汽动力循环、气体动力循环和制冷循环的分析计算。

- **第四部分**为化学反应系统的热力学原理，包括化学反应的热效应计算及赫斯定律和基尔霍夫定律、反应的最大功和化学平衡。

- 学生通过本课程的学习，掌握并运用热力学的基本规律，培养逻辑思维能力，从而进行热力计算和分析。为学习后继有关专业课程，从事能源动力工程的设计、管理和科学研究提供重要的热力学理论基础。

## 学习建议

=== "陈波"

    ## 工程热力学    91

    ### 任课老师： 孙志坚、刘少俊

    **课程：**首先，工热的课程安排确实有点怪，考的应用题主要是后面的部分，内容不多，但是课时比较多。期末占比60％；此外还有关于新型循环的课程展示、线上（学习通）的一些学习任务（而且看视频的学习任务居然鼠标都不能离开视频的界面，我还是找了一个学习通插件才勉强能挂在后台看课）、还有最后的课程论文。各种小的事情还是比较多的。

    **考试：**考试的话，前面有40分的简答题，一共8道，内容还是非常杂的，具体可以看一下98上其他同学的回忆卷，有些可能是线上学习的选择题。后面有60分的简答题，范围在热力学第二定律的计算、湿空气的计算、气体的流动、蒸汽动力循环、气体动力循环、制冷循环等等。后面的几个循环是肯定会出几个的，前面的可能是挑一个出。毕竟一共才三道大题，但是能出大题的内容还是非常多的。

    **学习：**日常的学习的话，szj的ppt我是确实看不下去，公式还有推导的过程很多都是贴图，但是老师板书的内容还是非常有用的，比老师的PPT有条理多了。后面的循环是lsj老师上课，还会进行展示。因为前面有40分简答题，所以要想拿高分的话，还是要好好看书或者是做一下线上的一些作业。后面的大题小于等于纸质作业的难度。
