# 动态建模理论及应用

## 课程内容

**动态建模这门课**

- 挺有用的控制入门课。让你熟悉MATLAB-SIMULINK的基础操作，熟悉火电机组的控制过程。课程内容是能源与环境系统工程原理及实践(六合一)的进阶版，包括气温控制系统、给水控制系统、燃烧控制系统、协调控制系统、脱硫脱硝控制系统等；还有一些动态建模技术的最新科研成果与应用。
- 老师属于好商量类型。作业太难太多都可以商量，不会的问题尽管询问助教。

**作业**

- 第一次作业：状态空间表达式
- 第二次作业：SCR脱销控制系统
- 第三次作业：主蒸汽温度控制
- 第四次作业：燃烧控制系统
- 第五次作业：协调控制系统

**考试**

- 平常作业的翻版（会让你手画控制流程图），好好复习作业题与PPT

### 课程大纲

无教材，基于PPT

- 第一章 **绪论**
- 第二章 **动态数学模型**
- 第三章 **智能控制技术**
- 第四章 **火电机组主要调节系统设计及仿真**
- 第五章 **综合能源负荷调度和闭环控制一体优化技术**
- 第六章 **基于动态模型的四角切圆锅炉烟温纠偏自动控制系统**
- 第七章 **火电机组大数据智能平台**
- 第八章 **熔盐储热系统自动控制方案的设计**
