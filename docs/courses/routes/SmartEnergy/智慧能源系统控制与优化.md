# 智慧能源系统控制与优化

## 课程内容

### 分数构成

1.平时成绩 40%(建模仿真作业、课堂表现)

2.课程实践大作业60%(书面报告、口头汇报、仿真结果)

（1）混合可再生能源发电系统仿真与优化

- 设计一个包含光伏、风电、储电的混合可再生能源发电系统
- 分别为光伏、风电、储电建立简化动态模型并在Simscape中仿真实现
- 分别为光伏、风电、储电建立基本控制系统并在Simscape中仿真实现
- 分析光能、风能波动条件下电子系统响应

（2）开放式选题，选择跟智慧能源系统相关(比如经济调度、综述等)

（3）完成方式

- 5~8分钟口头汇报
- 书面报告独立完成，单独评分
