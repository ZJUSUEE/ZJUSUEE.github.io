# 能源系统大数据分析理论与实践

## 课程内容

- 很好的机器学习入门课。包含数据预处理，人工神经网络等知识。
- 没有考试。几次小作业+大作业（团队合作，需要展示）
- 习题课会讲解小作业，助教老师人都很好，尽管问问题

### 分数构成

- 总成绩=平时成绩×40%十期终考试成绩×60%
- 其中平时成绩40%=出勤10%+报告30%
- 课堂提问:0-10分


### 课程大纲

1.能源与环境系统工程基础
2.能源的开发利用
3.污染物排放控制
