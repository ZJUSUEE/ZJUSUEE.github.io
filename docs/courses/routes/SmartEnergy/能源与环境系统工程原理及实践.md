# 能源与环境系统工程原理及实践

## 课程内容

**课程特色**

- 人送外号六合一，缺陷在内容参差不齐，安排过于杂乱。有些老师讲的东西还是非常值得一学的，算是能源专业的基础，在别的专业课也接触不到。有些老师上的甚至不如通识课。
- 由于老师流动性大，有的老师有布置作业，有的老师布置了作业也不会收，有的老师根本来走个过场。

**考试提示**

- 往年皆是开卷考，好好熟悉材料出分一定不差。
- 可以携带纸质材料（往年班级会集体打印PPT，涵盖所有答案）。建议为PPT材料制作目录，考试时翻页更快。

### 分数构成

**理论教学（75%）**	

- 期末考试（50%）
- 课后作业（25%）

**实践教学（25%）**	

- 上线率(5%)

## 学习建议

=== "王兰花秀丽"

    > 来源: https://www.cc98.org/topic/6086174

    ### 写在前面的建议
    1. 考试大概相当于50个填空(一题可能有多个空的那种),裸考可能会稍有压力,如果时间充裕建议看一下基础概念.
    2. 保底也要把各章节标题确定,不然考试时定位会花比较多时间.
    3. 仅作为个人参考的是,如果裸考可以先做后面的填空,题目知识点相对连续,选择题似乎有比较多的打乱和混合.

    ### 如果你没有时间/不打算看一遍所有ppt的话,我会建议你对速成关键词识别:
    1. 发电厂,燃煤蒸汽循环->热力发电
    2. 烟气脱硫,脱硝处理->硫/硝污染物脱除
    3. 多级/单极汽轮机,叶片,核汽轮机->汽轮机
    4. 智慧综合能源系统,数字孪生->综合能源系统
    5. 锅炉及其装置组件(过热器 再热器 空气预热器…),燃烧->锅炉原理
    6. 控制系统,控制->自动控制原理

    ### 个人感觉
    1. 颗粒物/污染物部分相对好找.
    2. 以及裸考的情况下燃煤蒸汽循环这个名词充满了干扰性(锅炉原理讨论了燃煤,燃烧计算等,汽轮机讨论了蒸汽及其循环,但看起来组合起来应该在热力发电找).
    3. 汽轮机题目总感觉有超越ppt,需要自己烧烤的知识,比较建议考前把握一下这里的基础概念.
    4. 综合能源系统比较考验图片识别能力.
    5. 锅炉原理知识点比较分散,加上页数最多.
    6. 自控在考卷上的选择题出现分布比较分散(感觉像是穿插在各模块之间,可能是错觉)
