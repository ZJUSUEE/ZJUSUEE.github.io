# 车辆工程

!!! info 

    浙大车辆以新能源与智能网联技术为核心，培养能设计汽车动力系统、研发智能驾驶算法、优化整车性能的复合型人才。核心课程包括**《汽车理论》《汽车构造》《设计与制造》《汽车动力系统原理》**，浙大车辆前身主要是**“内燃机”**专业，关注车辆动力系统的热力学基础，最近几年在**氢能动力**、**电池储能**、**智能控制算法**等研究领域也有所发展。

    目前车辆的课程资源较少，希望车辆的同学们能多多贡献！~
