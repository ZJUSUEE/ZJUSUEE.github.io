# 过程装备智能检测与控制技术（曾用名：过程装备控制技术）

## 课程内容

### 分数构成

课程成绩=卷面60%+平时40%
平时：出勤10分+作业50分（5章）+实验报告40分（4个）
实验包括4个演示实验、2个操作性实验、2个设计性实验
无期中考，每节课课前会点名提问上节课内容

## 学习建议

=== "此世之黯"
    这门课体验(在三门中)最好，我除了整理复习课就考前一晚上背作业题，然后考得最轻松。

    张志新老师讲课可能闷了点，但一对比就好很多（）理想中的科研人的样子。平时作业认真抄，就相当于整理复习资料了。这门课点名还挺多的

    复习课一定要听，怕智云没录到就开录音。张志新老师和曾胜老师都是复习课猛猛划重点的类型，被设备设计和过程机械洗礼了之后听这个要落泪了（）

    丢一份我弄的复习课整理当参考（）

    https://file.cc98.org/v4-upload/d/2025/0622/ys35c1r4.docx

=== "深夜读史记"

    - 开学之初张志新老师就把平时作业在xzzd上布置好了(前五章的部分课后题)，不过需要线下交作业本，老师上课用flash不用ppt
    - 作业90%都是简答题，其实就往前面翻几页就能找到答案那种，抄一抄有个印象就好，第一章第五章的几道计算题非常重要，大胆揣测应该每年都会考(求传递函数和PID算法表达式)
    - 实验要做两次，一次是反应釜和PID，一次是Arduino
    - 张志新老师的复习课仅用20min就把考试重点划完了，先是粗略过了下重点，然后是细化一下：第一题考你…，第二题考…，xxx我可以考你，但不是非得考你。期末也印证了那句经典台词“你们希望复习课长一点好还是短一点好，长一点你们就错了，要我肯定选择短的。短的话说明我讲的都是精华，没有干扰，相当于滤波给你们滤掉了”
    - 由于复习课的存在，期末的复习压力骤减，基本都是在面向对象的背背记记，比另两门轻松多了
  
    附个人整理复习提纲，供参考
