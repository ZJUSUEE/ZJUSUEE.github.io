# 过程设备设计

## 课程内容

### 分数构成

> 参考[深夜读史记](https://www.cc98.org/topic/6225566)


- 成绩组成：期末 60%，平时(作业+点名+实验报告) 40% (另两门成绩组成也一样)
- **期末半开卷**
- 课本有八章400多页，不过只上只考前四章的内容，其中2、4是重点(重中之重)
- 刘宝庆老师负责上前三章，李克明老师上第四章(第四章有100多页，内容跟前三章加起来差不多)
- 点名：这学期李克明老师共点过两次名，刚好期中考试周翘了没去，还都被点了…
- 作业：第二章第四章各布置了一次作业，线下交，类型为简答题+少量计算题
- 实验：5月份分组一次性做两个实验(内压&外压)：内压爆破和外压失稳

=== "此世之黯"

    今年由李克明老师负责后半的课程，我们班主任人品还行，就是讲课水平真不太行，尤其是复习课，题型也没讲导致很多人判断题没写错误原因，重点也没划，没说要带尺子，甚至智云没录到（不过无所谓），搜一搜就能看到骂他的帖（）想要补智云的可以再往前一届，刘宝庆老师讲的还是可以的

    由于是半开卷考试，相对来说压力没那么大，学长的那份pdf大部分还是找得到的，跟着刘老师的复习课过一遍方便找。还有就是作业题要重点复习，p131厚度的例题要重点复习，翻了整本书就这道例题方便考（虽然下一届好像要换教材了，就不好说了）比较难顶的还是没背到的作业题，没背到直接寄，分数还多，可以考虑把难背的修改进a4里。实验部分可以再改改理进去，今年要画设备图（干笑） 很复杂的公式其实可以直接删掉，基本没计算，但不敢删（）

=== "深夜读史记"
    
    **吐槽这门课的同学不少，包括但不限于**：

    - 要背记的东西巨多，堪比2-3个贡菜(工程材料)，哪怕一张A4弄成6版排满小字也根本不够
    - lkm老师的上课水平，尤其复习课质量欠佳(拖了一节课堂结果啥也没讲，重点不说考试题型也不说)
    - 期末试卷的恶心程度(判断题要写错误原因不写不给分，选择题搞不定项，简答题实验分析题要画一堆图，还要会推导公式，还有文字游戏)
    - 据了解，听说后续判断题没写错误原因也给分了，个人愚见纯shabi，卷子上白纸黑字写的错误原因没写 写错不得分，那看到题目的同学本来要打叉的由于不清楚错误原因也只能被迫打勾了，令人感慨

    **建议:** 

    - 可以翻看2023.06.15 刘宝庆老师的复习课智云，这b课不知道为什么近五年智云有且只有一节复习课
    - 最好自己根据ppt内容&回忆卷&作业题&前人A4，自己排一张A4出来(这样排的时候相当于复习了一遍，而且各块知识点的位置也会印象深点)
    - 考试时间比较紧张，故**A4要兼顾内容多少和字体大小**，字太小找起来费劲可能会写不完(今年0人提前交卷)，打印出来后也可以在空白处用其他颜色笔塞一些东西进去
    - 课本**P131的例题**应该是每年必考的，一定要放进A4里
